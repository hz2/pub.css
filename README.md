![Visualization of the codebase](./images/diagram.svg)

A simple CSS style toolkit that helps you build pages easily



Useage：

- For Less

```less
@import '~pub.css/less/index.less';
```

- For Scss

```scss
@import '~pub.css/scss/index.scss';
```

- css bundle

```scss
@import '~pub.css/css/index.css';
```

- main.js

```js
import 'pub.css/css/index.css';
```
